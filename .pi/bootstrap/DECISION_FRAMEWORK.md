# 决策框架

## 强制检查项（每个决策必须过一遍）

### 1. 风险定价检查
- [ ] 当前价格是否已经反映了已知风险？
- [ ] 市场在担心什么？这些担心是否过度？
- [ ] 如果利空出尽，上行空间有多大？

### 2. 周期位置检查
- [ ] 行业/商品周期处于什么阶段？（底部/上升/顶部/下降）
- [ ] 当前盈利是周期高点还是低点？
- [ ] PE高是因为盈利低谷还是估值泡沫？

### 3. 催化剂检查
- [ ] 未来1-3个月有什么可能改变基本面的催化剂？
- [ ] 财报/政策/行业事件/大宗商品价格？
- [ ] 这些催化剂是已知还是未知？

### 4. 仓位思维
- [ ] 不是买不买，是买多少？
- [ ] 试探性仓位（5-10%）vs 标准仓位（15-20%）vs 重仓（25-30%）
- [ ] 止损设在什么位置？亏损金额能否接受？

### 5. 反身性检查
- [ ] 如果买入后跌了10%，我会加仓还是止损？
- [ ] 如果买入后涨了10%，我会止盈还是持有？
- [ ] 提前想好退路，不被情绪左右

---

## 决策类型速查

| 情景 | 决策 |
|------|------|
| 估值极端便宜+周期底部+催化剂明确 | ✅ 重仓买入 |
| 估值合理+趋势向好+催化剂朦胧 | ✅ 标准仓位买入 |
| 估值偏贵+趋势不明+催化剂遥远 | ⏸️ 试探性仓位/观望 |
| 估值极端贵+趋势走坏+利空未出尽 | ❌ 回避 |
| 持仓浮亏>15%+基本面恶化 | ❌ 止损 |
| 持仓浮亏>15%+基本面完好 | ✅ 加仓（平均成本） |
| 持仓浮盈>30%+估值不再便宜 | ✅ 分批止盈 |

---

## 禁止的思维模式

❌ "PE太高所以不能买" → 改为："PE高是因为盈利周期低谷吗？如果是，真实PE是多少？"
❌ "主力在卖出所以不能买" → 改为："主力卖了多少了？卖完了吗？散户在接还是也在卖？"
❌ "基本面不好所以回避" → 改为："基本面在恶化还是触底？边际变化方向是什么？"
❌ "等回调再买" → 改为："什么价格买？如果不回调怎么办？"
