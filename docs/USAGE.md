# Investment Claude - AI 投资顾问使用指南

## 简介

Investment Claude 是一个基于 Claude Agent SDK 的 AI 投资顾问，专注于中国股市（A股和港股）的投资分析。

## 核心功能

### 1. 股票查询
- **实时价格**: 查询 A股/港股实时价格
- **基本信息**: 公司概况、行业、上市日期等
- **历史行情**: 日线/周线/月线历史数据

### 2. 财务分析
- **财务指标**: ROE、毛利率、净利率、负债率等
- **财务报表**: 资产负债表、利润表、现金流量表
- **质量评分**: 0-100分的基本面质量评分

### 3. 技术分析
- **技术指标**: MA、MACD、RSI、KDJ、CCI、布林带
- **价格形态**: 趋势判断、支撑阻力位
- **买卖区间**: 基于技术面和基本面的买入/止损/目标价

### 4. 估值分析
- **PE分位数**: 当前PE在历史中的位置（0-100%）
- **估值状态**: 低估/合理/高估判断
- **合理价值**: 基于格雷厄姆公式的估值

### 5. 市场数据
- **市场概览**: 上证、深证、创业板等指数
- **资金流向**: 北向资金、行业资金流、个股资金流
- **热门股票**: 百度热搜股票排行

### 6. 新闻资讯
- **个股新闻**: 东方财富、新浪财经、股吧热帖
- **市场新闻**: 财新、东财、百度财经日历
- **公司公告**: 业绩、分红、重组等公告

### 7. 机构数据
- **龙虎榜**: 游资和机构买卖明细
- **基金持仓**: 重仓该股的基金列表
- **股东信息**: 前十大股东、股东人数变化
- **融资融券**: 融资余额、融券余额
- **高管交易**: 内部人增减持

### 8. 股票筛选
- **按板块筛选**: 指定行业、ROE、PE等条件
- **质量筛选**: 按基本面质量评分筛选

### 9. 持仓管理
- **添加持仓**: 记录股票代码、数量、成本
- **查看持仓**: 查看所有持仓
- **删除持仓**: 移除持仓记录

## 可用的 Investment 工具函数

- `get_cash_flow` - 现金流量表

### 技术分析

### 估值分析

### 市场数据

## 技术架构

### 核心组件
1. **InvestmentTool** ([src/tools/InvestmentTool/InvestmentTool.tsx](src/tools/InvestmentTool/InvestmentTool.tsx))
   - 统一的投资工具接口
   - 调用 Python 桥接层获取数据

2. **Python Bridge** ([src/utils/python-bridge.ts](src/utils/python-bridge.ts))
   - Node.js 与 Python 的桥接层
   - 通过 spawn 调用 Python 脚本

3. **AKShare Bridge** ([python/akshare_bridge.py](python/akshare_bridge.py))
   - Python 数据获取脚本
   - 基于 akshare 库获取 A股数据
   - 基于新浪财经获取港股数据

### 数据流
```
用户提问 → AI 分析 → 调用 InvestmentTool
→ Python Bridge → akshare_bridge.py
→ 数据源 (akshare/新浪/东财) → 返回数据
→ AI 分析 → 生成报告
```

## 注意事项

### 数据时效性
- **股票价格**: 实时或 T+1（交易时间内实时，盘后为收盘价）
- **财务数据**: 季度报告（可能滞后 1-3 个月）
- **新闻数据**: 每日更新
- **技术指标**: 基于最新可用数据计算

### 风险提示
⚠️ **重要声明**
- 本工具提供的所有信息仅供参考，不构成投资建议
- 投资有风险，入市需谨慎
- 请根据自身风险承受能力做出投资决策
- 建议结合多方信息进行综合判断

### 数据来源
- A股数据: akshare (东方财富、新浪财经)
- 港股数据: 新浪财经、stooq
- 财务数据: 东方财富、新浪财经
- 新闻数据: 东方财富、新浪财经、财新、百度

## 常见问题

### Q: 为什么有些港股数据不可用？
A: 港股数据源有限，部分功能（如PE分位数、龙虎榜）仅支持A股。

### Q: 数据更新频率是多少？
A: 实时价格在交易时间内实时更新，财务数据随季报更新，新闻每日更新。

### Q: 可以分析美股吗？
A: 当前版本仅支持 A股和港股，美股支持计划中。

### Q: 如何添加自定义筛选条件？
A: 可以通过修改 Python 脚本中的筛选函数来添加自定义条件。
