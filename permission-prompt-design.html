  • 左右边距 20px，不贴边

  • 选项使用表格布局，对齐整齐

实现建议

  • 使用 borderStyle="double" 替代 "round"
  • 添加 marginX={2} 增加左右边距
  • 使用 inverse={true} 实现选中项反色
  • 添加 header 区域分离标题和工具名
  • 规则预览使用独立的 Box 包裹
  • 选项使用固定宽度实现对齐
  • 添加分隔线增强视觉层次
