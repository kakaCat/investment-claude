🛠️ 实现建议

权限提示改进：

  • 使用 borderStyle="double" 替代 "round"
  • 添加 marginX={2} 增加左右边距
  • 使用 inverse={true} 实现选中项反色
  • 添加 header 区域分离标题和工具名
  • 规则预览使用独立的 Box 包裹
  • 选项使用固定宽度实现对齐
  • 添加分隔线增强视觉层次

整体 TUI 改进：

  • 统一颜色方案：权限=黄色，AskUser=洋红，Plan=蓝色
  • 工具调用状态用颜色区分：成功=绿色，错误=红色，进行中=黄色
  • Thinking 默认折叠，节省空间
  • 输入框快捷键提示始终显示
  • 所有弹窗使用一致的边距和圆角
  • 增强视觉层次：分隔线、边框、阴影

交互改进：

  • 选中项反色高亮，一目了然
  • 快捷键提示清晰可见
  • 加载状态有明确的视觉反馈
  • 错误信息用红色突出显示
